- CSS `Media Query`를 사용하여 사이트의 크기와 구성이 달라지는 반응형구현.
- 상품을 더 쉽게 볼 수 있고 사용자의 접근성을 높일 수 있게 `Pagnation`을 적용.
- 메인페이지와 로그인페이지 그리고 2개의 상품페이지 총 `4페이지`.
- `Owl Carousel` 플러그인 사용.
